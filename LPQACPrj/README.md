# QAC 报告处理工具

快速删除 QAC 静态代码扫描报告中的指定 Severity 项。

## 快速开始

```bash
python process_all_html.py
```

## 使用示例

1. **运行脚本**
   ```bash
   python process_all_html.py
   ```

2. **查看可用的 Severity 值**
   ```
   找到以下Severity值: [1, 2, 3, 4, 5, 6, 7, 8]
   ```

3. **选择要删除的 Severity**
   ```
   请输入: 6,7,8
   ```

4. **确认删除**
   ```
   确认删除? (y/n): y
   ```

## 输入格式

- 单个: `6`
- 多个: `6,7,8` 或 `6 7 8`
- 全部: `all`
- 取消: `none` 或回车

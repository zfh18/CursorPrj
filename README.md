# CursorPrj

面向汽车电子开发的工具集合仓库，包含 DBC 合并、CANoe 测试模块同步、QAC 报告处理等脚本。

## 项目结构

- `MergeDBC/`：将多个 DBC 合并为单个 DBC（见 `MergeDBC/README.md`）。
- `AutomaticTest/Template/`：按 Excel 规则同步 CANoe 测试模块（见 `AutomaticTest/Template/README.md`）。
- `LPQACPrj/`：处理 QAC 扫描报告及相关注入脚本（见 `LPQACPrj/README.md`）。

## 说明

- 各子目录为相对独立的小工具，请优先查看对应目录下的 `README.md` 获取使用方法。
